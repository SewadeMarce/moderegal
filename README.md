# e-commerce Next.js

Projet e-commerce développé avec Next.js et TypeScript.

## Installation

```bash
npm install
```

## Démarrage

```bash
npm run dev
```

Ouvrez ensuite `http://localhost:3000` dans votre navigateur.

## Contenu du projet

- `app/` : pages et routes de l'application
- `components/` : composants réutilisables
- `lib/` : utilitaires et intégrations
- `public/` : ressources statiques
- `types/` : définitions TypeScript
- `script/` : scripts pour la base de données

## Notes

- Le projet utilise Next.js 14+ avec le nouveau système App Router.
- Adaptez les clés d'API et la configuration de paiement selon votre environnement.
